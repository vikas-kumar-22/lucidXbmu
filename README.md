# lucidXbmu -- Student--Psychiatrist Interaction Platform

lucidXbmu is a web-based platform designed to facilitate interaction
between students and psychiatrists. The system provides a structured
interface for managing information, handling interaction data, and
delivering a responsive user experience.

------------------------------------------------------------------------

## Features

-   Student--psychiatrist interaction interface
-   Appointment and data handling using JSON
-   Modular frontend with reusable components
-   Backend server using Node.js
-   Clean and responsive user interface

------------------------------------------------------------------------

## Tech Stack

**Frontend** - HTML - CSS - JavaScript

**Backend** - Node.js - JSON-based storage

## How to Run

Clone the repository:

``` bash
git clone https://github.com/vikas-kumar-22/lucidXbmu.git
cd lucidXbmu
```

Install dependencies:

``` bash
npm install
```

Run the server:

``` bash
node server.js
```

Open browser:

    http://localhost:3000
